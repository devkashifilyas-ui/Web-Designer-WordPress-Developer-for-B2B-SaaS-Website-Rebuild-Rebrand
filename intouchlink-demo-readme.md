# InTouchLink SaaS Website Redesign Concept

This repository contains a **SaaS marketing website concept demo**
created for the InTouchLink platform.\
The purpose of this prototype is to demonstrate how the product can be
repositioned as a **modern multi‑industry community operations
platform** rather than a niche senior living tool.

The demo focuses on **UX clarity, SaaS positioning, and conversion
optimization** for demo bookings.

------------------------------------------------------------------------

# Demo Overview

The prototype is a **single HTML page** designed to illustrate the
homepage of a modern B2B SaaS website.

Key goals of the redesign:

-   Reposition InTouchLink as a **multi‑industry SaaS platform**
-   Improve **conversion flow for demo bookings**
-   Modernize the visual design to match SaaS products like Stripe,
    Notion, and HubSpot
-   Simplify navigation and information architecture
-   Demonstrate modular product capabilities

------------------------------------------------------------------------

# Sections Included in the Demo

The homepage prototype includes the following sections:

1.  Hero Section\
    Platform value proposition with dashboard preview and demo CTA.

2.  Product Modules\
    Core platform capabilities including:

    -   Digital Signage / Community TV
    -   Community Portal & Messaging
    -   Activity & Event Management
    -   Menu & Meal Tracking
    -   Service Requests & Integrations

3.  Industries\
    Demonstrates how the platform serves multiple sectors:

    -   Healthcare
    -   Education
    -   Hospitality
    -   Corporate
    -   Residential

4.  How It Works\
    Three step onboarding model for the platform.

5.  Testimonials\
    Social proof showing operational benefits and engagement
    improvements.

6.  Pricing Model\
    Example SaaS pricing tiers with feature differentiation.

7.  Demo Conversion Section\
    Primary lead generation CTA encouraging visitors to book a demo.

# Technology Used

The demo is intentionally lightweight and deployable anywhere.

Technologies:

-   HTML5
-   CSS3
-   Google Fonts (Inter + Syne)
-   Vanilla JavaScript
-   Responsive layout

No frameworks or build tools are required.

------------------------------------------------------------------------

# How to Run the Demo

Option 1 -- Local preview

1.  Download the file `intouchlink-demo.html`
2.  Open it in any browser

Option 2 -- Instant deployment

You can deploy the demo instantly using Netlify:

1.  Go to https://app.netlify.com/drop
2.  Drag and drop `intouchlink-demo.html`
3.  Netlify generates a live public URL
